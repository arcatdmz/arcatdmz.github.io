おうちハック (Realmote)

2011-2024
関連キーワード: NUI
ツイート
概要なめこカメラ
'12-13
リズム家電
'12
Realmote
'11-12

概要

スマートフォンやタブレット端末のWebブラウザから、電灯やエアコン、自動掃除機ルンバなどの家電製品を遠隔操作できるようにした。

開発環境

  • Java + Phybots on Mac OS X
  • パソコン用学習リモコン PC-OP-RS1
  • ルンバ用Bluetoothインタフェース RooTooth

  • ブログ記事: Realmoteで家のリモコンを一つにまとめる
