プログラミングって何だっけ？

ブラウザのコンソールに入力してみよう:

1+1;

もう少し複雑な例:

window.alert("Hello World!");

こんなこともできる:

var follow = function (tw) { tw.widgets.createFollowButton("arcatdmz", document.body).then(function (el) { el.style.position = "absolute"; el.style.top = "50%"; el.style.left = "50%"; el.style.transform = "scale(5)"; el.style.zIndex = 1000; }); }; if (window.twttr) { follow(window.twttr); } else { var src = document.createElement("script"); src.src = "https://platform.twitter.com/widgets.js"; window.twttr = { _e: [ follow ] }; document.body.appendChild(src); }

※プログラミング環境の歴史について詳しく知りたい場合は環これ(統合開発環境これくしょん / IDE Collections)を読んでみてください。リンク先は、加藤の博士論文から関連研究の章の一部を抜粋して和訳したものです。
