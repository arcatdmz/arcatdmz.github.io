IntelliViewer
Hierarchical Source Code Viewer on Adaptive Keyboard

2010
Keywords: IDE, NUI

Abstract

This is a user interface for helping navigation of source code in the integrated development environment. For each row of the keyboard, available namespaces, classes, member variables and methods are shown on the keytops. The user interface works on the prototype adaptive keyboard developed by Microsoft Applied Science Group.

This work was done in collaboration with lab colleagues and presented at UIST 2010 Student Innovation Contest.
