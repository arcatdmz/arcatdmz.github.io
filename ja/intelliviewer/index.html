IntelliViewer
Hierarchical Source Code Viewer on Adaptive Keyboard

2010
関連キーワード: IDE, NUI
ツイート

概要

キートップの最下列から順に、列ごとにネームスペース、クラス、メンバ変数とメソッドの一覧を表示し、統合開発環境においてソースコード内のナビゲーションを高速化するユーザインタフェースです。Microsoft Applied Science GroupがAdaptive Hardwareプロジェクトで開発したAdaptive Keyboardのプロトタイプ上で動作します。

研究室の有志による共同プロジェクトで、UIST 2010のStudent Innovation Contestに参加しました。
